# Multi-Ralph Project

## Overview

This is a multi-package project managed with pnpm workspaces.

## Project Structure

```
ralph/
├── packages/     # Individual packages/modules
├── scripts/      # Build and utility scripts
└── .claude/      # Claude Code configuration and skills
```

## Development

### Prerequisites

- Node.js (v18+)
- pnpm

### Getting Started

```bash
pnpm install
```

## Commands

- `pnpm build` - Build all packages
- `pnpm test` - Run tests
- `pnpm lint` - Lint code
